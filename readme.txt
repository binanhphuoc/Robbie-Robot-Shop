To compile, follow these steps:

	make

To run program:
	
	./exe

By default, the program loads data from file "data.txt" and saves to the same file when exiting.

To delete all objects and executable:

	make clean

An executable file "exe" produced by the program has been included in the "src" folder.


FEATURES

- Create new parts and new models with custom window, making it easy to have an overview of information and easy to pick a desired part.
- View Robot Parts and Robot Models with browser. On click of an item in the browser, Details and Image of each item appears on the right.
- Currently, login window has been created
- Create new order
